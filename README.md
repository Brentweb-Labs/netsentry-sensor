# NetSentry Sensor (`netsentry-sensor`)

NetSentry Sensor is an open-source, high-throughput network security monitoring agent built entirely in Rust. It functions as an intelligent edge sensor designed to intercept, parse, and analyze edge-network traffic with near-zero latency, passing structured threat telemetry up to the NetSentry Cloud Management Console.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Language: Rust](https://img.shields.io/badge/language-Rust-orange.svg)](https://www.rust-lang.org/)

---

## Key Features

* **High-Performance Rust Core:** Leverages low-overhead async execution routines via Tokio for packet and log ingestion.
* **Native Suricata Engine Integration:** Continuously consumes and monitors `eve.json` event structures generated by deep-packet inspection engines.
* **Outbound WireGuard Tunneling:** Initiates secure outbound transport layers back to the management cluster—allowing full remote operation from behind complex NAT arrays without editing your perimeter firewall rules.
* **Fail-Open System Resilience:** Designed as a transparent network bridge (`br0`) that defaults to non-blocking pass-through states if runtime services are ever interrupted.
